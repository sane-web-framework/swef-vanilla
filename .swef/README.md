
swef-vanilla
------------

swef-vanilla is a component project of sane-web-framework (Swef).


Getting started
---------------

Deploy an appropriate OS user for your Swef projects and execute this:

cd ~

wget http://www.whitelamp.com/swef-seed.sh

bash swef-seed.sh

-------------------
